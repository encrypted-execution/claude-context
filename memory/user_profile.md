Archis Gore (me@archisgore.com) is a security researcher and the author of:
- The Encrypted Execution whitepaper at www.encrypted-execution.com
- USPTO patent 10,733,303 (pledged to the public domain)
- The Polyverse Polymorphic Linux product (former CTO/founder)
- The PHP / WordPress polyscripting reference implementations at
  github.com/encrypted-execution and github.com/archisgore/polyscripted-wordpress

He treats "scrambling" / "polyscripting" / "encrypted execution" as a
coherent body of work: the same closure-based diversity-as-defense thesis
applied at progressively lower layers (lexer tokens → bytecode → calling
convention → kernel ABI → custom ISA).

Collaboration style:
- Expects clear-eyed threat modeling, not marketing. Honest "this is a
  diversity defense, not a confidentiality boundary" framing is welcome.
- Asks for parallel agent teams for research-heavy questions and is
  comfortable with multi-dossier outputs.
- When pausing work, expects state to be saved durably to the repo so
  the next session (his or an agent's) can pick up cold.
